Linear Regression with Multiple Features

  • Linear Regression with Multiple Features

The above figure shows a system that would be suitable for linear regression with multiple features. We need to fit a linear hypothesis, in the case of 2 features it will be a plane, in the case of more features it will be impossible to visualize yet it will be linear algebraicly. We will have the following variables defined. We will have $$m, n, x^{(i)}, x_j^{(i)}$$ as number of data, number of features, array of ith feature and value of the feature j in ith sample. Our hypothesis this time will be: $$h(x) = \theta_0 + \theta_1 x_1 + \dots + \theta_n x_n$$ We will define the following column vectors in order to make easy operations.

$$ x = \begin{bmatrix} 1 \\ x_1 \\ x_2 \\ \dots \\ x_n \end{bmatrix}, \theta = \begin{bmatrix} \theta_0 \\ \theta_1 \\ \theta_2 \\ \dots \\ \theta_n \end{bmatrix}$$ Then we can define simply $$ h(x) = \theta^T x$$

The cost function in this case will be denoted just the same as the cost function of the single feature linear regression. $$ J(\theta) = \frac{1}{2m}\sum_{i=1}^{m} (h(x^{(i)})-y^{(i)})^2$$ As the idea of gradient descent is exactly the same, we will have the derivative of the cost function as $$\frac{\partial J}{\partial \theta_j} = \frac{1}{m}\sum_{i=1}^{m}(h(x^{(i)})-y^{(i)})x_j^{(i)}$$ From which we have the final algorithm for gradient descent with multiple features: $$ \theta_j := $$ $$ \theta_j - \alpha \frac{1}{m}\sum_{i=1}^{m}(h(x^{(i)})-y^{(i)})x_j^{(i)$$ repeated until convergence.
