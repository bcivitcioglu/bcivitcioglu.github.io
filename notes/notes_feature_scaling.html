Feature Scaling

  • Feature Scaling

For example let us assume that our data set consists of two features. $$ x_1 \in [ 0, 4000], x_2 \in [1,5]$$ This is the idea of feature scaling. We want them to be in the same scale so that our algorithm can work better. If we draw the cost function without scaling, we will get the eliptic version. With the appropriate scaling, we will get the circle version. It will make the steps less complicated to calculate.

We will have $$ x_j^{scaled} = \frac{x_j^{(i)}}{max{(x_1)}}$$ For all the values in each feature.

Ideally it is a good idea to put them all into the range: $$ [-1,1]$$

Mean Normalization

In mean normalization we have a simple formula too. This case we will be scaling with respect to the average of all the values of a feature. $$x_j^{scaled} = \frac{x_j-average(x_j)}{max(x_j)-min(x_j)}$$

Remark

In order to make sure that the gradient descent is working, we can plot the cost function. We expect the cost function to decrease over time. We can declare convergence if the cost function decreases less than some small number we fixed.
