This is the very basic architecture of a single neuron. In this picture, we will be calling the output layer, a neuron. A neuron will receive inputs from the previous layer, and it will make the inputs go through an activation function. The activation function can be chosen among many functions. For example we can go with the sigmoid activation function, hyperbolic tangent function or more commonly used, ReLU function. $$ h(x) = ReLU(x) = max(0,x) $$ or $$ h(x) = \sigma(x) = \frac{1}{1+e^{-\theta^T x}}$$

This is a neural network with more complicated architecture. The layers between the input layer and the output layer are called hidden layers. We will define some terminology before we go deeper: $$a_i^{(j)}$$ is the activation of unit i in layer j. For example the second layer third unit's activation function would be denoted as $$a_3^{(2)}$$ And moreover, we have the notion of weights. Weight matrices will be denoted as $$\theta^{(j)}$$ being the weights of maps from layer j to layer j+1. We will denote the input layers with x's, and hidden layers with a's. Note that there is a bias unit: $$x_0 = 1$$ Let us give an example calculation of the layers of this neural network. For instance if we want to calculate the second layer first unit, we will do the following operation: $$ a_1^{(2)} = $$ $$ \sigma(\theta_{10}^{(1)}x_0 + \theta_{11}^{(1)}) x_1 + \theta_{12}^{(1)} x_2 + \theta_{13}^{(1)} x_3)$$ and similarly the ouput layer will be: $$ a_1^{(3)} = $$ $$ \sigma(\theta_{10}^{(2)} a_0^{(2)} + \theta_{11}^{(2)} a_1^{(2)} + \theta_{12}^{(2)} a_2^{(2)} + \theta_{13}^{(2)} a_3^{(2)})$$ Therefore we can see that in order to calculate the output layer, we need all the values of the weights and all the values of the nodes of the hidden layers. Note that if the network's number of units in layer j is $$s_j$$ then we have the matrix $$\theta^{(j)} \rightarrow {s_j \times s_{j+1}}$$

Therefore in order to calculate the hypothesis, we need to calculate all the values of the nodes. They are defined by activation functions which depend on the weights. Therefore the thing we need to calculate here is the weights. Once we have all the weights, then we can succesfully calculate the hypothesis in the output layer. The way we calculate the weights is with forward and backward propagation.
